# Scan 02 — 4% bullish

## 概要

当日に強い上昇を見せ、かつ相対強度（RS）と出来高の裏付けがある銘柄をフィルタリングするスキャン。
「今日、モメンタムが明確に発火した銘柄」を即時検出する。
ブレイクアウト初日やボラタイルな上昇日に有効なスキャンであり、ウォッチリストへの緊急追加候補を生成する。

**ソース:** `src/scan/rules.py` → `_scan_bullish_4pct()`

---

## 条件一覧

| # | 条件 | 閾値 | 参照列 | config キー |
|---|------|------|--------|-------------|
| 1 | Relative Volume | >= 1.0x | `rel_volume` | `relative_volume_bullish_threshold` |
| 2 | 当日変化率 | >= 4.0% | `daily_change_pct` | `daily_gain_bullish_threshold` |
| 3 | 始値からの上昇率 | > 0.0% | `from_open_pct` | — |
| 4 | Raw RS21 | > 60.0 | `raw_rs21`（fallback: `rs21`） | — |

---

## 各条件の計算式

### rel_volume（Relative Volume）
```python
avg_volume_50d = volume.rolling(50).mean()
rel_volume     = volume / avg_volume_50d
```
当日出来高を過去50日平均出来高で割った比率。1.0 = 平均と同等。

---

### daily_change_pct
```python
daily_change_pct = close.pct_change() * 100.0
# = (today_close - yesterday_close) / yesterday_close * 100
```
前日比の終値変化率（%）。

---

### from_open_pct
```python
from_open_pct = (close - open) / open * 100.0
```
当日の始値から終値への変化率（%）。
正の値 = 始値より高い終値（引けにかけて強い）。

---

### raw_rs21（Raw RS — 21日窓）
```python
# ステップ1: 銘柄/SPY の価格比率時系列を算出
ratio = ticker_close / spy_close

# ステップ2: 直近 21 日分の ratio を抽出
window = ratio.tail(21)

# ステップ3: その 21 点の中でのパーセンタイルランク（0〜100）
raw_rs21 = percentile_rank(window).iloc[-1]
```
銘柄自身の**過去 21 日の ratio 時系列内**での現在位置のランク。
外部銘柄との横断比較ではなく、自己比較によるランク。

`raw_rs21` 列が存在しない場合は `rs21` にフォールバックする（`_raw_rs()` ヘルパー）。

---

## このスキャンでフィルタリングされる特性

### 通過する銘柄の特性

**1. 当日に強い上昇（>= 4%）**
4% 以上の上昇という明確なモメンタムの発火を要求する。
小幅上昇（1〜3%）の銘柄はノイズとして排除され、明確なブレイクアウトや強い買い圧力が発生した銘柄のみ通過する。

**2. 出来高の裏付けあり（Relative Volume >= 1.0）**
平均以上の出来高を伴う上昇であることを確認する。
出来高を伴わない "空吹き" や流動性の低い銘柄での跳ねを除外し、機関投資家参加の可能性がある動きを選別する。

**3. 引けにかけて強い（from_open_pct > 0%）**
始値より終値が高いことを要求する。
寄り天（始値が高くその後下落）の銘柄を除外する。「買いが続いた一日」の銘柄のみ通過する。
ギャップアップ後に失速して始値を割り込んだ銘柄も除外される。

**4. 直近 21 日での相対強度が上位（Raw RS21 > 60）**
自身の過去 21 日 ratio 時系列の中での現在位置が上位 40% 以上であることを要求する。
「今日だけ急騰した」弱い銘柄を除外し、直近1ヶ月を通じて相対的に強い状態が続いている銘柄を選別する。

### 除外される銘柄の典型パターン

| パターン | 除外される理由 |
|----------|----------------|
| 出来高を伴わない急騰 | rel_volume < 1.0 |
| 上昇幅が軽微（< 4%） | daily_change_pct < 4.0 |
| 寄り天・ギャップアップ失速 | from_open_pct <= 0% |
| 直近の相対強度が弱い | raw_rs21 <= 60 |

---

## 7リストとの対応

このスキャン結果は **List 4: 4% Gainers** としてそのまま流用される（daily_change_pct >= 4.0% の条件のみ）。
RS・出来高・from_open 条件は 4% Gainers リストには含まれない。
