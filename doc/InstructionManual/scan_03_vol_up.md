# Scan 03 — Vol Up

## 概要

平均を大きく上回る出来高を伴いながら当日に上昇した銘柄をフィルタリングするスキャン。
「今日、異常な量の買いが入った」銘柄を広く拾い上げる最もシンプルなモメンタム検出スキャン。
4% bullish と異なり上昇幅に閾値を設けないため、出来高増加を伴う初動の銘柄も含めて広くカバーする。

**ソース:** `src/scan/rules.py` → `_scan_vol_up()`

---

## 条件一覧

| # | 条件 | 閾値 | 参照列 | config キー |
|---|------|------|--------|-------------|
| 1 | Relative Volume | >= 1.5x | `rel_volume` | `relative_volume_vol_up_threshold` |
| 2 | 当日変化率 | > 0.0% | `daily_change_pct` | — |

---

## 各条件の計算式

### rel_volume（Relative Volume）
```python
avg_volume_50d = volume.rolling(50).mean()
rel_volume     = volume / avg_volume_50d
```
当日出来高を過去50日平均出来高で割った比率。
閾値 1.5x は「平均の 1.5 倍以上の出来高」を意味する。

> 4% bullish の閾値（1.0x）より厳しい 1.5x を要求する点に注意。
> 上昇幅の条件を緩めた代わりに、出来高の要求水準を引き上げている。

---

### daily_change_pct
```python
daily_change_pct = close.pct_change() * 100.0
```
前日終値比の変化率（%）。`> 0.0` のみ要求するため、0.1% の微小な上昇でも通過する。

---

## このスキャンでフィルタリングされる特性

### 通過する銘柄の特性

**1. 出来高が平均の 1.5 倍以上（Relative Volume >= 1.5）**
平常時の 50% 増という明確な出来高の異常を要求する。
通常の取引日に比べ、有意に多い参加者が存在することを示す。
IR リリース、アナリストアップグレード、業界ニュース、ETF リバランス等のカタリストを持つ銘柄が多く含まれる。

**2. 当日上昇（> 0%）**
出来高増加の方向が「買い超過」であることを確認する。
売り超過による出来高急増（ディストリビューション）を除外する。
上昇幅に上限・下限を設けないため、緩やかな上昇から急騰まで幅広くカバーする。

### 除外される銘柄の典型パターン

| パターン | 除外される理由 |
|----------|----------------|
| 平均的な出来高の上昇 | rel_volume < 1.5 |
| 出来高増加を伴う下落 | daily_change_pct <= 0% |
| フラットな出来高増加（横ばい） | daily_change_pct = 0% |

### 4% bullish との違い

| 項目 | Vol Up | 4% bullish |
|------|--------|------------|
| 上昇幅 | > 0%（制限なし） | >= 4% |
| Relative Volume 閾値 | >= 1.5x | >= 1.0x |
| from_open 条件 | なし | > 0% |
| RS 条件 | なし | Raw RS21 > 60 |
| 目的 | 出来高異常の広範検出 | 強いモメンタム発火の検出 |

Vol Up は「上昇幅は小さくても出来高が異常に多い銘柄」も捕捉できる。
ブレイクアウト前日の出来高先行、または大口の静かな買い集めを早期発見するのに適している。
