# 全スキャン定義（簡潔版）

`doc/SystemDocs/Scan` を参照して、全スキャンを「何を検出するか」で短くまとめた一覧です。  
（詳細条件や厳密な閾値は各 `scan_*.md` を参照）

## 有効スキャン（デフォルト有効）

| Scan名 | 簡潔な定義 |
|---|---|
| `4% bullish` | 当日の上昇率が大きく、出来高も伴う強い陽線を検出する。 |
| `Momentum 97` | 週次・四半期の騰落率ランクがともに非常に高い、強モメンタム銘柄を検出する。 |
| `97 Club` | 総合スコアと短期RSがともに最上位級のリーダー銘柄を検出する。 |
| `Pocket Pivot` | 直近期間の出来高ピークを上回る陽線で、かつ50日線上にいる当日需要を検出する。 |
| `PP Count` | 直近20日で Pocket Pivot が規定回数以上出ている銘柄を検出する。 |
| `Weekly 20% plus gainers` | 直近1週間で20%以上上昇した急騰銘柄を検出する。 |
| `Near 52W High` | 終値が52週高値近辺にあり、かつ総合スコアが一定以上の銘柄を検出する。 |
| `Three Weeks Tight` | 週足で3週間の値動きが小さく収縮し、さらにVCS条件を満たす銘柄を検出する。 |
| `RS Acceleration` | 短期RSが中期RSを上回り、短期RS自体も高水準な銘柄を検出する。 |
| `VCS 52 High` | ボラ収縮とRS条件を満たしつつ、52週高値からの乖離が小さい銘柄を検出する。 |
| `VCS 52 Low` | ボラ収縮とRS条件を満たしつつ、52週安値帯に位置する銘柄を検出する。 |
| `Volume Accumulation` | 上昇日の出来高優位と相対出来高の高さから、需給の蓄積がある銘柄を検出する。 |
| `Pullback Quality scan` | 上昇トレンド中の「浅すぎず深すぎない」押し目と出来高クールダウンを検出する。 |
| `Reclaim scan` | 押し目後に21EMAを終値で再び上抜くリクレイム局面を検出する。 |
| `Sustained Leadership` | 短期・中期・長期のRSがそろって高い、継続的リーダー銘柄を検出する。 |
| `Trend Reversal Setup` | 反転初期の土台（50日線回復、RS条件、PP実績など）がそろった銘柄を検出する。 |
| `Structure Pivot` | 構造的なLL->HL形成後、ピボット価格を当日に初回ブレイクした銘柄を検出する。 |
| `21EMA Pattern H` | 21EMA高値帯付近を保ったまま前日高値を更新する、強い押し目継続型を検出する。 |
| `21EMA Pattern L` | 21EMA下側を一度試して戻し、前日高値更新につながる防衛型を検出する。 |

## 無効スキャン（実装はあるがデフォルト無効）

| Scan名 | 簡潔な定義 |
|---|---|
| `21EMA scan` | 週次リターン・終値位置・移動平均帯位置から、21EMA周辺の上昇基調銘柄を広く検出する。 |
| `Vol Up` | 上昇日に相対出来高が高い銘柄をシンプルに検出する。 |
| `VCS` | VCSスコアが閾値以上の銘柄を検出する。 |
