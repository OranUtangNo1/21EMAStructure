# Start OraTek

Start the OraTek CLI from the repository root.

## Workflow

1. Use the repository root `C:\reository\21EMAStructure`.
2. Use the project virtual environment only: `.\.venv\Scripts\python.exe`.
3. Do not use system Python.
4. Start the CLI with:

```powershell
.\.venv\Scripts\python.exe -m src.cli.oratek
```
